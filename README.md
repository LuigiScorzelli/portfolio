# Portfolio Landing Page

Landing page personale per presentare progetti di automazione AI, integrazioni CRM e applicazioni full-stack orientate a casi d'uso reali.

## Descrizione

Il sito presenta servizi, stack tecnico, progetti demo e contatti professionali.

E' sviluppato come sito statico con HTML, CSS e JavaScript vanilla, con una versione italiana e una versione inglese.

## Obiettivo

Mostrare in modo chiaro il mio posizionamento come sviluppatore specializzato in automazioni AI, workflow n8n, integrazioni CRM e prototipi full-stack pensati per processi reali.

## Stack

- HTML5
- CSS3
- JavaScript
- Responsive design
- Deploy statico

## Struttura

```text
.
├── index.html      # Versione italiana
├── en.html         # Versione inglese
├── styles.css      # Stili principali
├── script.js       # Menu mobile e logica base
└── assets/         # Immagini e asset
```
